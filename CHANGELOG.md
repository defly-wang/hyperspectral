# 版本变化日志

## [Unreleased]

### 2026-03-23

### 新增功能
- **USGS光谱库支持**
  - 预览模块支持直接浏览USGS光谱库
  - 支持按类别/子类别筛选光谱
  - 子类别名称中英文双语显示（矿物、植被、液体等）
  - 训练模块支持直接加载USGS光谱库目录作为训练数据
  - 数据清洗模块支持USGS .txt文件验证和解析
  - 识别模块支持USGS光谱文件识别

- **波长范围调整**
  - 所有模块波长最低值从400nm调整为350nm
  - USGS光谱读取支持根据仪器类型(ASD/BECK/NIC4/AVIRIS)选择对应波长文件

### 2026-03-18

### 新增功能
- **预览模块伪彩功能改进**
  - 多图片水平排列时子图间距设为0
  - 隐藏子图边框，实现"融合"效果
  - 点击图片时在画面顶部中间显示文件名

- **识别模块改进**
  - 文件列表改为表格形式（文件名、识别结果、置信度三列）
  - 识别完成后自动将结果填入表格对应行

### 2026-03-17

### 新增功能
- **多语言支持**
  - 支持简体中文和英文两种语言
  - 菜单 Language 可随时切换语言
  - 切换语言时实时更新所有模块界面

- **数据清洗模块**
  - 无效数据检测：空值、NaN、无穷值、负值、超出范围值、数据长度不匹配、方差过少
  - 异常数据检测：IQR和Z-Score两种方法
  - 重复数据检测：基于相关系数的光谱相似度检测
  - 异种类别数据检测：检测与整体趋势差异大的异常光谱
  - 文件格式检查：支持格式验证、表头检查、编码检测
  - 无效文件检查：文件读取失败、数据为空、过滤后无数据、数据点过少
  - 报告导出功能
  - 问题列表点击预览光谱曲线

- **数据集分割模块**
  - 支持两种分割模式：训练/验证/测试集 或 训练/测试集
  - 可配置训练集、验证集、测试集比例（百分比形式，默认70%/15%/15%）
  - 分割进度条显示
  - 分割前自动清除旧文件（带确认提示）
  - 支持数据打乱和随机种子设置
  - 后台线程执行，避免UI卡顿

- **模型训练模块改进**
  - 支持加载预分割的数据目录（train/val/test结构）
  - 训练后自动使用验证集进行验证
  - 显示验证集准确率和分类报告

- **识别模块**
  - 加载训练好的模型进行预测
  - 批量识别多个文件
  - 显示识别结果及置信度

- **应用名称**
  - 更名为"高光谱数据处理系统"

### 功能改进
- 波长过滤从500nm改为400nm，后又调整为350nm
- 文件列表按文件名排序

### 性能优化
- 使用 python-calamine 替代 openpyxl，大幅提升Excel读取速度
- 模型训练采用多进程并行加载数据
- 数据清洗模块改用 python-calamine
- 加载文件时每100个文件更新一次进度显示

### UI改进
- 标签页：Preview / Data Cleaning / Data Split / Model Training / Recognition
- 新增数据菜单：数据清洗、数据分割
- 新增模型菜单：训练模型、加载模型、识别
- 新增语言菜单：中文/English
- 预览面板代码重构为独立的 PreviewPanel 类
- 预览面板左侧区域固定最大宽度，防止因文件路径过长导致变宽
- 光谱图图例可拖动
- 光谱曲线支持鼠标点击选中/取消选中，选中曲线高亮加粗
- 伪彩色图像支持多选显示，每列一个图像水平排列
- 数据清洗模块光谱预览使用 SpectrumPlotWidget，共享曲线选中功能
- 应用图标

## [1.0.0] - 2026-03-11

### 新增功能
- **数据查看功能**
  - 支持读取 `.isf` 格式高光谱数据文件
  - 支持读取 `.xlsx/.xls` 格式高光谱数据文件
  - 光谱曲线可视化显示（波长 vs 反射/透射率）
  - 多文件选择支持（Ctrl+多选、Shift+连续选择）
  - 多光谱曲线对比显示

- **伪彩色图像**
  - 将光谱数据转换为伪彩色图像展示

- **数据预处理**
  - 平滑滤波：Savitzky-Golay 滤波、移动平均滤波
  - 归一化：Min-Max 归一化、Z-Score 标准化
  - 基线校正：AirPLS、基线扣除
  - 自动过滤 500nm 以下波长数据

- **模型训练**
  - Random Forest 分类器
  - SVM 分类器
  - Gradient Boosting 分类器
  - 训练数据自动从目录加载（按子目录分类）
  - 训练结果准确率显示
  - 详细分类报告（Precision、Recall、F1-Score）
  - 模型保存功能（.pkl 格式）

### UI/UX 改进
- 基于 PyQt6 的现代化桌面界面
- 菜单栏：文件、视图、模型、帮助
- 标签页切换：Preview / Model Training
- 状态栏显示当前操作状态
- 快捷键支持（Ctrl+O、Ctrl+T、Ctrl+Q）

### 项目初始化
- 项目结构搭建
- PyInstaller 打包配置
- requirements.txt 依赖管理
